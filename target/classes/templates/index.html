คุณสมบัติ

เพิ่มสมาชิก (Create)

เพิ่มสมาชิกใหม่เข้าสู่ระบบ เช่น การลงทะเบียนสมาชิกใหม่ เพื่อให้สมาชิกสามารถเข้าถึงสิทธิ์และข้อมูลต่าง ๆ ที่เกี่ยวข้องกับระบบได้ทันที.

ดูข้อมูลสมาชิก (Read)

เรียกดูข้อมูลของสมาชิกที่มีอยู่ในระบบ เช่น ชื่อสมาชิก ข้อมูลการติดต่อ หรือสถานะการใช้งาน เพื่อให้ผู้ดูแลระบบสามารถตรวจสอบและจัดการข้อมูลได้อย่างง่ายดาย.

แก้ไขข้อมูลสมาชิก (Update)

ปรับปรุงหรือแก้ไขข้อมูลสมาชิกที่มีอยู่ในระบบ เช่น การเปลี่ยนแปลงที่อยู่ หรือปรับปรุงข้อมูลการติดต่อ เพื่อให้ข้อมูลสมาชิกเป็นปัจจุบันและถูกต้องอยู่เสมอ.

ลบสมาชิก (Delete)

ลบสมาชิกที่ไม่ต้องการออกจากระบบ เช่น สมาชิกที่ยกเลิกการใช้งาน หรือสมาชิกที่ไม่มีการใช้งานเป็นระยะเวลานาน โดยข้อมูลของสมาชิกที่ถูกลบจะไม่สามารถกู้คืนได้.

Objective

วัตถุประสงค์

โปรเจกต์ "Spring Boot Thymeleaf CRUD Pagination Sorting Web Application" นี้ถูกพัฒนาขึ้นเพื่อสร้างระบบจัดการข้อมูลที่สามารถดำเนินการเพิ่ม, อ่าน, แก้ไข, และลบ (CRUD) ได้อย่างครบถ้วน พร้อมทั้งมีฟีเจอร์การจัดการการแบ่งหน้า (pagination) และการจัดเรียงข้อมูล (sorting) ที่ช่วยให้สามารถจัดการข้อมูลขนาดใหญ่ได้อย่างมีประสิทธิภาพ โดยใช้ Thymeleaf ในการสร้างส่วนติดต่อผู้ใช้ (UI) ที่ใช้งานง่ายและรองรับการใช้งานบนอุปกรณ์หลากหลายชนิด สำหรับส่วนของ Frontend ระบบนี้ใช้ HTML และ CSS ร่วมกับ Bootstrap Framework เพื่อออกแบบ UI ที่ตอบสนองต่อทุกอุปกรณ์ และใช้ SweetAlert สำหรับการแสดงผลการแจ้งเตือน (alert) ที่สวยงามและเข้าใจง่าย.
